# Python_num_guessing_game
<h1>CLI Number Guessing Game created using Python</h1>


<p>A simple command-line number guessing game built using Python.
The program generates a random number between 1 and 100, and the user must guess it.</p>
<br>
<h2>Concepts Used :</h2><br>
<p>while loop<br>
try and except (error handling)<br>
random.randint()<br>
Conditional statements (if-elif)<br>
User input handling.<br>
<br>
<h2>How to Run..</h2>
<br>
<p>Make sure Python is installed.<br>
Save the file as number_guessing_game.py<br>
Run the program!:python number_guessing_game.py</p>
