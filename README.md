# Python_num_guessing_game
CLI Number Guessing Game created using Python
Number Guessing Game (Python CLI)
A simple command-line number guessing game built using Python.
The program generates a random number between 1 and 100, and the user must guess it.

Concepts Used
while loop
try and except (error handling)
random.randint()
Conditional statements (if-elif)
User input handling

How to Run..

Make sure Python is installed.
Save the file as number_guessing_game.py
Run the program!
